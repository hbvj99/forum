API doc

You first need API Key to access or change any content. Make sure you are registered user and you have username and password.
Here are some of the examples using curl.

Obtaining token key

REQUEST:
curl -X POST http://localhost:8000/api-token/ -d username="YOUR_USERNAME" -d password="YOUR_PASSWORD"

RESPONSE:
{"token":"77737d18c29e195709nxx76hy7b15dxxxxxxxxxx"}

Discussion

Get discussion post

REQUEST:
curl -X GET http://localhost:8000/api-discussion/DISCUSSION_TITLE/ -H "Authorization: Token YOUR_TOKEN_KEY"

Create discussion

img, tags are optional. Find your user ID in profile.

curl -d '{"user_id":USER_ID, "title":"TITLE_NAME","content":"YOUR_CONTENT", "category":"CATEGORY_NAME"}' -H "Content-Type: application/json" -H "Authorization: Token YOUR_TOKEN_KEY" -X POST http://localhost:8000/api-discussion/new/

Edit discussion post

img, tags are optional.

curl -d '{"user_id":USER_ID, "title":"TITLE_NAME","content":"YOUR_CONTENT", "category":"CATEGORY_NAME"}' -H "Content-Type: application/json" -H "Authorization: Token YOUR_TOKEN_KEY" -X PUT http://localhost:8000/api-discussion/list/

For image;

curl -X PUT -F 'user_id=USER_ID' -F 'title=TITLE_NAME' -F 'content=YOUR_CONTENT' -F 'img=@/FILE_PATH.EXTENSION' -H "Authorization: Token YOUR_TOKEN_KEY" -H 'content-type: multipart/form-data' http://localhost:8000/api-discussion/DISCUSSION_TITLE/

Delete discussion post

curl -X DELETE http://localhost:8000/api-discussion/DISCUSSION_TITLE/ -H "Authorization: Token YOUR_TOKEN_KEY"
